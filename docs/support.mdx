## Dedicated Account Manager

> Available on **Enterprise** and **Dedicated** plans.

Enterprise customers receive a dedicated Technical Account Manager for:

- White-glove onboarding and setup
- Best-practice configuration guidance
- Ongoing operational support
- Custom check and compliance policy development

## Security Assessment

We maintain a comprehensive set of internal [security policies and documentation](/security) that address key requirements for GDPR and SOC 2 compliance.

For Enterprise and Dedicated customers, we support your security assessment process with custom responses and documentation as needed.

Please check out the [Security & Compliance overview](/security/index) for more details.
